MI PRIMER PAGINA CON Java Script

Ejemplo con JS donde muestra un aviso cuando se carga la pagina


puede ser de dos maneras:

la primera de manera interna, donde el script este en el HEAD, se escribe de esta forma: /**/ y ahí se carga junto con la pagina


la segunda de manera externa, donde script va en una carpeta con el nombre JS, dentro de esta se crea un archivo con extensión js ejemplo: /*alert("Hola Mundo con Java Script")*/ ;y en el HEAD se escribe: /**/ y ahí va a aprecer primero el msj del script y cuando hacemos clic en aceptar, se carga el resto de la pagina
