EJERCICIO PRACTICO

Ejercicio Practico: ingresar dos numeros mediante un prompt y almacenar sus valores en dos variables distintas. Realizar una resta entre los dos vaolores en javascript. Almacenar el resultado en una variable. Importar el archivo JS desde una pagina html. Si el resultado de la resta es mayor a 0, mostrar medianre un console.log() o un alert() el mensaje "es mayor a 0". Si es mayor a cero, comparar si el numero es par, en caso que los sea, mostrar un mensaje "es par" en caso de que no uno que diga "es impar". En caso que sea menor a 0, mostrar un mensaje que diga "es menor o igual a 0"
